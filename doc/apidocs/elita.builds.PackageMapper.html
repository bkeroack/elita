e.b.PackageMapper : class documentation

Part of elita.builds View In Hierarchy

Applies supplied package map to the build. Assumes pre-validated package map and a build_dir that exists

This puts the generated package files directly in the build storage directory and returns a mapping of package names to filenames/types (but doesn't update the build object)
